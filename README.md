# 🎂 Calculadora de Idade

Aplicação web que calcula a idade do usuário a partir da data de nascimento informada

---

## 🚀 Tecnologias

* HTML
* CSS
* JavaScript

---

## ✨ Funcionalidades

* Cálculo automático da idade
* Atualização instantânea do resultado
* Interface responsiva e intuitiva

---

## 🎯 Objetivo

Projeto desenvolvido para praticar lógica de programação e manipulação de datas em JavaScript
